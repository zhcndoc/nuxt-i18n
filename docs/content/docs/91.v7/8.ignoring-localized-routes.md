---
title: 忽略本地化路由
---

::callout{icon="i-heroicons-light-bulb"}
此功能不支持 `no_prefix` [策略](./strategies)。
::

如果您希望某些页面仅在某些语言中可用，可以配置支持的语言列表以覆盖全局设置。这些选项可以在页面组件内或全局的模块选项中指定。

### 选择本地化路由

::code-group

```js {}[pages/about.vue]
export default {
  nuxtI18n: {
    locales: ['fr', 'es']
  }
}
```

```js {}[nuxt.config.js]
i18n: {
  parsePages: false,
  pages: {
    about: {
      en: false,
    }
  }
}
```

::

### 禁用本地化路由

::code-group

```js {}[pages/about.vue]
export default {
  nuxtI18n: false
}
```

```js {}[nuxt.config.js]
i18n: {
  parsePages: false,
  pages: {
    about: false
  }
}
```

::
